# 70304_display_strings_va_list

## Build and install

```console
mkdir build
cd build
cmake ..
cmake --build .
cmake --install .
```

## Run

```console
$ ./dist/bin/demo --help
Usage: demo
   Demonstrate a function for printing a variable number of strings.
$ ./dist/bin/demo
aaa
bbb cccc
dddd eee ff
Special Agent Dale Cooper
```
