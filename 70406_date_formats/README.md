# 70406_date_formats

## Build and install

```console
mkdir build
cd build
cmake ..
cmake --build .
cmake --install .
```

## Run

```console
$ ./dist/bin/now --help
Usage: now
    Display the current time in a few formats.

$ ./dist/bin/now
Wednesday, September 18, 2024  05:30pm
Wed, 18 Sep 24  17:30
09/18/24  5:30:29 PM
```
